# 24 — Контент: все растения и редкие культуры

**Фаза:** 6 — Контент · **Зависит от:** 02, 08

## Цель

Заполнить `plants.json` всеми растениями по стихиям + редкими культурами с особыми
правилами роста.

## Состав (GDD «Растения по стихиям» + «Редкие культуры»)

- **Огонь:** Огнецвет (15м), Лавовая Ягода (30м), Жаркий Мох (45м).
- **Вода:** Морской Цветок (20м), Жемчужный Мох (40м), Паровая Орхидея (2ч).
- **Земля:** Корень Силы (25м), Каменный Плод (1ч), Древесный Гриб (3ч).
- **Гроза:** Грозовой Перец (20м), Электрический Трюфель (2ч), Молниевый Кактус (4ч).
- **Лёд:** Ледяная Роза (30м), Снежный Гриб (1.5ч), Хрустальный Иней (3ч).
- **Нейтральные:** Лунный Лотос (2ч, любой биом), Теневой Мак (4ч, теневой сад).
- **Воздух:** Воздушный Одуванчик (15м), Перо-трава (40м), Облачный Хлопок (2ч).
- **Природа/Свет:** Цветок Жизни (20м), Сладкий Корень (45м), Солнцецвет (3ч).
- **Редкие культуры:** Сердце Кометы (24ч, не ускоряется, семена из экспедиций),
  Цветок Времени (8ч, −30м инкубации), Астральная Лоза (12ч, космический сад,
  остров 4), Ночной Кристалл (6ч, теневой сад, остров 3).

## Что делать

1. **Заполнить `plants.json`** по каждому: id, name, element, `grow_time_minutes`,
   `native_biome`, `sprite`, `description` (из GDD). Растёт в родном биоме ×1.5
   (логика в task-08, не в данных).
2. **Редкие культуры — особые флаги.** Добавить поля: `rare: true`, `boostable:
   false` (Сердце Кометы не ускоряется ничем), `seed_source` ('expedition'|'shop'|
   'quest'), `min_island`. Учесть в task-08 (запрет ускорения) и в посадке (доступно
   ли семя/биом/остров).
3. **Эффекты при сборе.** Цветок Времени собранный → −30м к инкубации (task-10);
   Сердце Кометы/Астральная Лоза — ингредиенты космических/легендарных рецептов
   (task-23).
4. **Сверка.** Все растения, упомянутые в рецептах (task-23), существуют здесь.

## Definition of Done

- [ ] Все растения и 4 редкие культуры в `plants.json` с корректными полями.
- [ ] Редкие культуры с особыми правилами (не ускоряются / только в своём биоме/
      острове / семена из источника) ведут себя верно.
- [ ] Каждое растение из рецептов скрещивания присутствует.
