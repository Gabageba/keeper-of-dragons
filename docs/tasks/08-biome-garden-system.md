# 08 — Биомы и сад: посадка / рост / сбор

**Фаза:** 2 — Сад · **Зависит от:** 05, 03

## Цель

Вместо обычных грядок — магические биомы-сады. Биом 3×3 = 9 слотов (улучшение до
4×4 = 16). В каждый слот сажается растение, оно растёт по таймеру, собирается в
склад. В родном биоме растение растёт в 1.5× быстрее.

## Модель данных

`GardenState { biome, slots: PlantSlotState[] }`, `PlantSlotState { plant, planted_at }`
(уже в `src/types`). Множитель скорости роста хранить не нужно — считается:

```
growMs = plantDef.grow_time_minutes × 60000
        / (nativeBiomeMatch ? 1.5 : 1)
        / (altarBonus ? 1.05 : 1)        // алтарь в центре биома +5%
        / sezonarHarvestBonus ...        // глобальные бонусы (task-22)
ready = planted_at + growMs
```

## Что делать

1. **Создание биома.** Строится за ресурсы стихии (GDD: Вулканический = 200
   Жар-Угля + 5 Живой Глины и т.д.). Занимает 3×3 на сетке (task-05).
2. **Сцена/панель сада.** Тап по биому → сетка слотов. Пустой слот → выбор семени
   из доступных (фильтр по тому, что разблокировано/куплено). Растущий слот →
   таймер и стадия роста (3 кадра: росток → бутон → готово). Готовый → тап = сбор.
3. **Рост от timestamp.** Никаких per-frame таймеров: стадия = функция от
   `(now - planted_at) / growMs`. Пересчёт при заходе и раз в N секунд.
4. **Сбор.** Готовое растение → `addResource(plantId, n)` (растения — тоже
   «ресурс»-инвентарь для кормления и скрещивания), XP, освобождает слот.
5. **Улучшение 3×3 → 4×4.** За ресурсы. Центральная клетка может стать алтарём
   стихии (+5% скорости роста, но −1 слот).
6. **Засев пачкой.** Кнопка «засеять всё выбранным» — удобство (Hay Day-style).
7. **Эликсир Роста / Живая Роса.** Разовый ускоритель −50% времени (task-20).

## Definition of Done

- [x] Огнецвет в Вулканическом саду растёт за ~10 мин (15 / 1.5), вне его — 15 мин.
- [x] Стадии роста отображаются и переживают перезагрузку (от timestamp).
- [x] Сбор кладёт растение в инвентарь и освобождает слот.
- [x] Улучшение до 4×4 добавляет слоты; алтарь даёт +5% и убирает 1 слот.
