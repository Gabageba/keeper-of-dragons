## Что делать

1. **`data/buildings.json`.** Для каждого: id, размер, функция, `unlock_level`,
   уровни апгрейда с ценой и эффектом (`level: { cost, effect }`).
2. **`BuildingSystem`.** Состояние построек на острове (через task-05 placements +
   уровень). `upgrade(uid)` — проверка ресурсов/уровня игрока, повышение уровня,
   применение эффекта.
3. **Эффекты апгрейдов (хуки в системы):**
   - Инкубатор → лимит слотов (task-10);
   - Гнездо → множитель производства (task-06);
   - Зал Скрещивания → число слотов + множитель времени (task-11);
   - Биом → 9↔16 слотов (task-08); Верфь → слоты экспедиций (task-19);
   - Кузница → очереди (task-20).
4. **Меню строительства.** Список доступных (по уровню) зданий с ценой → режим
   размещения (task-05). Заблокированные показывать с требуемым уровнем.
5. **Скин гнезда.** Облик гнезда уникален по виду дракона (строка 🏠 в Бестиарии) —
   подставлять спрайт по `dragon.id` (task-27).

## Definition of Done

- [ ] Все здания строятся (по уровню) и размещаются на сетке.
- [ ] Апгрейд списывает ресурсы и реально меняет поведение системы (слоты/скорость).
- [ ] Заблокированные здания показывают нужный уровень.
- [ ] Гнездо использует скин по виду дракона.
