# 12 — Вылупление и взросление детёныша

**Фаза:** 3 — Скрещивание · **Зависит от:** 10, 09, 13

## Цель

Из готового яйца вылупляется детёныш. Детёныш не производит ресурсы — его нужно
покормить 3–5 раз, прежде чем он станет взрослым и займёт гнездо. Новый вид
записывается в Книгу с наградой.

## Жизненный цикл дракона

```
egg → (hatch) → baby → (feed ×N) → adult
```

`DragonState.stage` ∈ `'egg' | 'baby' | 'adult'`, `feedings` — счётчик кормлений.

## Что делать

1. **Вылупление.** Кнопка «Вылупить» на готовом яйце:
   - создать `DragonState { uid, id, stage:'baby', feedings:0, parent_ids }`;
   - анимация: яйцо трясётся → трещина → появляется детёныш (task-28);
   - **если вид новый** → `useGameStore.getState().discoverInBook(id)`, показать карточку открытия
     с лором и наградой (task-13, task-16).
2. **Размещение детёныша.** Нужно свободное гнездо (или временный «детский уголок»).
   Привязать детёныша к гнезду на сетке (task-05/06).
3. **Взросление кормлением.** Каждое кормление (task-09) повышает `feedings`. При
   `feedings >= growThreshold` (3–5, баланс task-26) → `stage = 'adult'`. Между
   кормлениями — кулдаун (нельзя скормить всё сразу). Опционально — таймер
   `grow_time_minutes` как альтернативный путь.
4. **Визуал стадий.** Детёныш меньше/милее, у взрослого — полный спрайт и аура
   редкости. Прогресс взросления — индикатор над гнездом.
5. **Только взрослый.** Производит ресурс (task-06), участвует в скрещивании
   (task-11) и экспедициях (task-19).

## Definition of Done

- [ ] Готовое яйцо вылупляется в детёныша по тапу.
- [ ] Новый вид записывается в Книгу один раз, с наградой и карточкой лора.
- [ ] Детёныш не даёт ресурсов, требует N кормлений до взрослого.
- [ ] После N кормлений становится взрослым и начинает производство.
- [ ] Состояние стадии/кормлений переживает перезагрузку.
