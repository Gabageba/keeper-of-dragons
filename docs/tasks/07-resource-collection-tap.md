# 07 — Окно ухода за драконом: сбор, кормление, улучшение

**Фаза:** 1 — Ядро фермы · **Зависит от:** 06

## Цель

Тап по гнезду открывает **окно ухода** (`DragonFeedModal`), где игрок управляет
производственным циклом: забирает готовую партию, кормит дракона и улучшает выход
ресурса. Это «сочная» обратная связь, ради которой игрок возвращается.

## Что делать

1. **Тап по гнезду → окно.** Тап по гнезду с драконом открывает `DragonFeedModal`
   (через `openDragonPanel`); пустое гнездо — обычное меню постройки. Авто-сбора по
   тапу больше нет — всё через окно.
2. **Сбор готовой партии.** В состоянии READY — кнопка «Собрать» →
   `collectResource(uid)`: переносит `pending_amount` в склад (с возможным переливом
   за `MAX_RESOURCE_PER_TYPE`), сбрасывает партию → дракон снова HUNGRY.
   - анимация: иконки ресурса летят к счётчику HUD, звук «дзынь» (task-29);
   - немного XP за сбор (task-16).
3. **Производство в процессе.** В состоянии PRODUCING — прогресс-бар и обратный
   отсчёт до готовности.
4. **Кормление.** В состоянии HUNGRY — список растений из инвентаря с пометками
   «🟢 любит / 🔴 не любит» и предпросмотром размера партии (`→ N`). Тап →
   `feedDragon(uid, plantId)`. В прочих состояниях список заблокирован.
5. **Улучшение выхода.** Кнопка «Улучшить» тратит монеты (`yieldUpgradeCost(level)`),
   поднимает `yield_level` (+`YIELD_UPGRADE_BONUS_PER_LEVEL` к выходу), потолок —
   `YIELD_UPGRADE_MAX_LEVEL`.
6. **Сбор «всё сразу».** Кнопка «Собрать всё» (HUD) пробегает гнёзда в состоянии
   READY (`collectAllNests` → `NestSprite.isReady`).
7. **Тап-реакция дракона / идл-анимация** (task-28) — базовый хук здесь.

## Definition of Done

- [x] Тап по гнезду открывает окно ухода.
- [x] «Собрать» переносит готовую партию в склад (с переливом за лимит).
- [x] Кормление доступно только когда дракон голоден; в инвентаре виден предпросмотр N.
- [x] Улучшение выхода тратит монеты и растит партию.
- [x] «Собрать всё» собирает все готовые гнёзда.
