# 07 — Сбор ресурсов и тап-реакции дракона

**Фаза:** 1 — Ядро фермы · **Зависит от:** 06

## Цель

Тап по гнезду собирает накопленный ресурс (летит в склад с приятной анимацией),
а сам дракон реагирует на тап — подпрыгивает/радуется. Это «сочная» обратная связь,
ради которой игрок возвращается.

## Что делать

1. **Сбор по тапу.** Тап по «пузырю»/гнезду с накопленным ресурсом — через колбэк
   сцены в стор `useGameStore.getState().collectResource(dragonUid)` (начисляет
   накопленное и ставит `last_collected = now`); базовая логика уже в `dragonsSlice`;
   - анимация: иконки ресурса летят к счётчику в HUD (tween по дуге), счётчик
     «тикает» вверх, звук «дзынь» (task-29);
   - начислить немного XP за сбор (task-16).
2. **Тап-реакция дракона.** Тап по самому дракону (без накопления или просто так):
   дракон подпрыгивает (`scale`/`y` tween), моргает, издаёт звук. Это из GDD
   «реакция на тап». Радость = маленький эмоут «❤».
3. **Сбор «всё сразу».** Кнопка «Собрать всё» в HUD — пробегает по всем гнёздам
   с накоплением. (Бонус Кометара — «мгновенный сбор всех» раз в день — частный
   случай этого.)
4. **Анти-даблтап.** Защита от двойного начисления (флаг «собирается»/блокировка на
   время анимации).
5. **Идл-анимация.** Когда не трогают — лёгкое покачивание/дыхание (task-28),
   но базовый хук завести здесь.

## Definition of Done

- [ ] Тап по полному гнезду переносит ресурс в склад, пузырь гаснет.
- [ ] Счётчик ресурса в UI растёт с анимацией.
- [ ] Дракон визуально реагирует на тап (подпрыгивает).
- [ ] Нельзя собрать дважды один и тот же запас.
- [ ] «Собрать всё» работает по всем гнёздам.

## Ссылки на GDD

- «Визуальный стиль» → Анимации (реакция на тап, happy-эмоция).
- «Кометар» — мгновенный сбор всех ресурсов раз в день.
