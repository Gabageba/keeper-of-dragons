# 14 — Рынок Совета: заказы и квесты

**Фаза:** 4 — Цель и мотивация · **Зависит от:** 03, 08

## Цель

Заказы Совета — основной источник монет и фрагментов карты. Игрок сдаёт ресурсы/
растения/сплавы по заказу и получает награду. Плюс линейная цепочка сюжетных квестов,
обучающая механикам.

## Модель данных

```ts
interface OrderDef {
  id: string;
  tier: 'simple' | 'medium' | 'hard' | 'legendary';
  requires: { item: string; amount: number }[];   // ресурсы/растения/сплавы
  reward: { coins?: number; gems?: number; fragments?: number; egg?: string;
            resources?: Record<string, number> };
  min_level?: number;
}
```

## Что делать

1. **MarketScene (модалка).** Показывает 5 активных заказов. Каждый — список
   требований с прогрессом (есть X из Y), кнопка «Сдать» (активна при наличии всего).
2. **Генерация заказов.** Рынок обновляется 3 раза в сутки (каждые 8 ч). Пул
   заказов фильтруется по уровню игрока; награда масштабируется с уровнем. Слот
   обновляется по таймеру от timestamp (как всё остальное).
3. **Сдача.** Списать требуемые предметы → выдать награду (монеты/фрагменты/яйцо/
   камни) → XP → заменить заказ новым (или оставить слот пустым до обновления).
4. **Ручное обновление.** Можно сбросить заказ за камни (по желанию).
5. **Сюжетные квесты.** Отдельная линейная цепочка (`quests.json`): «Первое яйцо»,
   «Знакомство с садом», «Экспериментатор», «Путешественник», «Хранитель
   Архипелага», «Тайна Некзуса». Каждый квест = условие + награда + следующий шаг;
   служит туториалом. Отслеживать через события стора (task-03).
6. **Тиры наград (GDD):** Простой ~150 монет; Средний 500 + 3 фрагмента; Сложный
   1500 + редкое яйцо; Легендарный 5000 + Яйцо Хроновиса.

## Definition of Done

- [ ] Рынок показывает 5 заказов, обновляется каждые 8 ч (от timestamp).
- [ ] «Сдать» списывает предметы и выдаёт награду только при полном комплекте.
- [ ] Награды-яйца попадают в Инкубатор (task-10).
- [ ] Сюжетная цепочка ведёт игрока по базовым механикам.
