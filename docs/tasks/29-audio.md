# 29 — Звук и музыка

**Фаза:** 7 — Арт и полировка · **Зависит от:** 07 (точки вызова)

## Цель

Звуковое сопровождение через Phaser Sound Manager (Web Audio): фоновая музыка по
островам/настроению + SFX на ключевые действия. Минимум зависимостей.

## Что делать

1. **Загрузка аудио** в PreloadScene (`this.load.audio`). Форматы `.ogg` + `.m4a`/
   `.mp3` fallback для iOS Safari. Сжать (низкий битрейт для мобилок).
2. **Менеджер звука `src/systems/AudioManager.ts`.** Обёртка над Phaser Sound:
   `playSfx(key)`, `playMusic(key, {loop})`, громкость master/music/sfx, mute.
   Настройки сохраняются (task-04).
3. **SFX на действия:** сбор ресурса «дзынь», тап по дракону (милый писк), посадка/
   сбор растения, вылупление яйца, успешное скрещивание/новый вид (фанфары), сдача
   заказа, постройка/апгрейд, открытие острова, ошибка/недоступно.
4. **Музыка по контексту:** спокойная тема фермы (своя на остров — вулканическая/
   морская/грозовая/космическая), отдельная для карты, торжественная для финала.
   Кросс-фейд при смене сцены.
5. **Мобильная специфика.** Аудио стартует только после первого тача (политика
   автоплея). Пауза музыки при сворачивании приложения (`visibilitychange`).
6. **Настройки звука** в меню (task-30): слайдеры music/sfx, кнопка mute.

## Definition of Done

- [ ] Музыка играет и меняется по островам/сценам (с кросс-фейдом).
- [ ] Ключевые действия озвучены, без задержки/наложений.
- [ ] Аудио корректно стартует после первого касания (iOS/Android).
- [ ] Настройки громкости/mute работают и сохраняются.
- [ ] Музыка ставится на паузу при сворачивании.
