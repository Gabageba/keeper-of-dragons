# 19 — Экспедиции (Верфь)

**Фаза:** 5 — Прогрессия и мир · **Зависит от:** 06, 18

## Цель

Пассивный способ добыть редкие ресурсы, яйца и Фрагменты Карты. Игрок отправляет
взрослого дракона в экспедицию на фиксированное время — он возвращается с наградой.
Простая петля без микроменеджмента.

## Модель данных

```ts
interface ExpeditionRouteDef {
  id: string;
  name: string;
  duration_hours: number;
  best_element: Element | 'any';          // повышает шанс редкой награды
  rewards: { item: string; amount: number; chance: number }[];
  min_slot_level?: number;
}
interface ExpeditionProgress {
  route_id: string; dragon_uid: string; started_at: number; ready_at: number;
}
```

## Что делать

1. **`data/expeditions.json` (GDD «Маршруты»):**
   - Вулканические руины (2 ч, Огонь/Земля) → ресурсы + редко яйцо Лавакрона;
   - Коралловый лабиринт (3 ч, Вода/Лёд) → ресурсы + редко Жемчужный Мох ×10;
   - Грозовое ущелье (4 ч, Гроза) → ресурсы + Фрагменты Карты ×3;
   - Небесные острова (6 ч, любой гибрид) → Астральная Лоза + редко Сердце Кометы;
   - Временная Аномалия (12 ч, Астралон) → Яйцо Хроновиса (20%) + Осколки.
2. **UI Верфи.** Список маршрутов, время, рекомендуемая стихия, таблица наград.
   Выбор дракона (взрослый, свободный) → запуск. Дракон занят весь срок.
3. **Шанс редкой награды.** Если стихия дракона совпадает с `best_element` —
   повышенный шанс (множитель к `chance`). Бонус Древнего Тёмного: +25% к редким
   наградам экспедиций. Бурелом: −10% к таймерам экспедиций.
4. **Завершение от timestamp.** `ready_at <= now` → выдать награды (бросок по
   `chance`), вернуть дракона. Работает «задним числом» после оффлайна (task-04).
5. **Слоты.** Верфь ур.1 — 1 слот, ур.2 — 2, ур.3 — 3 (task-18).
6. **Яйца-награды** идут в Инкубатор (task-10); Фрагменты — в открытие островов
   (task-17); особые яйца (Хроновис, Древо-семя) — только отсюда.

## Definition of Done

- [ ] Запуск занимает дракона на время маршрута; по возвращении выдаёт награды.
- [ ] Совпадение стихии повышает шанс редкой награды.
- [ ] Награды-яйца/фрагменты корректно попадают в нужные системы.
- [ ] Число параллельных экспедиций ограничено уровнем Верфи.
- [ ] Завершение корректно после оффлайна.
