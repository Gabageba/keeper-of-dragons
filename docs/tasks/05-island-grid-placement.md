# 05 — Сетка острова и размещение построек

**Фаза:** 1 — Ядро фермы · **Зависит от:** 01, 03

## Цель

Остров — сетка тайлов (клетка 64×64 px). Здания и биомы занимают прямоугольные
области. Игрок размещает/перемещает их перетаскиванием с проверкой занятости и
запретом ставить на воду/скалы.

## Модель данных

```ts
interface IslandDef {
  id: string;            // 'ashen'
  name: string;
  cols: number; rows: number;          // напр. 14×11 (Пепельный)
  blocked: [number, number][];         // вода/скалы (нельзя строить)
  unlocked_cells: [number, number][];  // открытая земля (расширяется за ресурсы)
}
interface Placement {
  uid: string;
  type: string;          // 'nest' | 'incubator' | 'biome_volcanic' | ...
  refId?: string;        // ссылка на dragon uid / garden и т.п.
  x: number; y: number;  // левый-верхний угол в клетках
  w: number; h: number;  // размер в клетках
}
```

Размеры построек (GDD «Сетка острова»): Гнездо 2×2, Инкубатор 2×2, Зал Скрещивания
3×2, Рынок 2×2, Библиотека 2×3, Кузница 3×2, Картостол 2×2, Храм 3×3, Зал
Хранителя 3×3, Декор 1×1. Биом-сад 3×3 (→ улучшение 4×4), Космический 4×4.

## Что делать

1. **Слой сетки.** `src/systems/GridSystem.ts`: хранит занятость клеток (2D-массив
   или `Set<"x,y">`), методы `canPlace(x,y,w,h)`, `place(p)`, `remove(uid)`,
   `worldToCell(px,py)`, `cellToWorld(x,y)`.
2. **Отрисовка острова.** Тайлы травы/воды/скал (плейсхолдер-графика, потом
   спрайты task-27). Органичная форма острова = `blocked` по краям.
3. **Режим размещения.** Из меню «Построить» берём здание → полупрозрачный призрак
   следует за пальцем по клеткам (snap to grid) → зелёный, если `canPlace`, красный
   если нет (пересечение/вода/закрытая земля). Тап «✔» — ставим, списываем ресурсы.
4. **Перемещение/снос.** Долгий тап по зданию → режим «переместить/убрать».
5. **Расчистка земли.** Закрытые клетки открываются за ресурсы (кнопка на завале).
   Постепенное открытие территории = стимул прогрессии (GDD).
6. **Камера.** Drag-pan + pinch-zoom (для мобилок), clamp к границам острова.
   Phaser `cameras.main.setBounds`.

## Definition of Done

- [ ] Нельзя поставить здание на воду/скалу или поверх другого.
- [ ] Призрак подсвечивается зелёным/красным корректно.
- [ ] Здание сохраняется в `SaveData` (позиция в клетках) и восстанавливается.
- [ ] Камера панорамируется и зумится в пределах острова.
- [ ] Расчистка закрытой земли работает и стоит ресурсов.
