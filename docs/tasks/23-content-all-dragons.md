# 23 — Контент: все 57 драконов и рецепты

**Фаза:** 6 — Контент · **Зависит от:** 02, 11, 13

## Цель

Наполнить `dragons.json` всеми 57 видами и `breeding.json` всеми рецептами по
дереву скрещивания из GDD. Контент — только данные, без правки кода.

## Что делать

1. **Заполнить `dragons.json`** по каждому виду: id, name, element, rarity, resource,
   `production_per_hour` (по таблице редкостей: общ.8 / необ.12 / редк.16–20 /
   эпич.24–30 / легенд.35–45 / мифич.50–70 / древн.80–120), `favorite_food`/
   `disliked_food`, `favorite_biome`/`disliked_biome` (Бестиарий; адаптивные —
   `null` + `adaptive:true`), таймеры инкубации/взросления, `global_bonus` (для
   эпиков+), `bio` (из Бестиария), `sprite`.
2. **Заполнить `breeding.json`** всеми парами родителей → результат (см. «Родители»
   в полном списке). Указать `required_plants`, `boost_plant`, `breed_time_hours`
   (по редкости результата), `chance`, `fail_result`.
   - Особые: Хроносаур требует Хроновис (экспедиция) + Цветок Времени ×5;
     Древо-Дракон — Древо-семя (экспедиция); Некзус — Этериал + Кометар + Сердце
     Кометы ×3. Прародитель — финальный квест; Вечный — 100% Книги (не через рецепт).
3. **Сверка ссылок.** Все `result`/`parent`/`plant` существуют (dev-валидация task-02).
4. **ID-конвенция.** Латиница, snake_case, стабильные (используются в сейвах).

## Definition of Done

- [ ] 56 выводимых видов + Вечный заведены в `dragons.json` с полными полями.
- [ ] Все рецепты из GDD в `breeding.json`, проходят dev-валидацию.
- [ ] Производство соответствует таблице редкостей.
- [ ] Особые рецепты (экспедиционные яйца, квест, 100%) обрабатываются корректно.
