# 17 — Карта архипелага: острова, туман войны, навигация

**Фаза:** 5 — Прогрессия и мир · **Зависит от:** 01, 03, 05

## Цель

Единая карта мира — экран выбора острова и индикатор прогресса освоения. Острова —
узлы-кнопки (открыт/закрыт), пути между ними открываются за Фрагменты Карты +
ресурсы. Центральное Сердце открывается последним.

## Что делать

1. **MapScene.** Фон — стилизованная морская карта на пергаменте (task-27). Узлы
   островов на фиксированных позициях (кольцо вокруг Сердца, см. GDD-схему). Пути
   между узлами — пунктир (золотой = открыт, тусклый с замком = закрыт).
2. **Состояния узла.** `locked` (силуэт + «?», под «туманом войны»-оверлеем) /
   `unlocked` (живой анимированный остров). Из `unlocked_islands` в `SaveData`.
3. **Открытие острова/пути.** Условие: `N` Фрагментов Карты (из экспедиций/заказов)
   + ресурсы текущего уровня + мин. уровень игрока. По оплате — снять туман
   (анимация), добавить в `unlocked_islands`.
   - Цены (GDD): О2 — 10 фрагментов + 500 Жар-Угля + 200 Живой Глины; О3 — 20 +
     1000 смеш.; О4 — 40 + 50 Звёздной Пыли; О5 — 60 + 5 Осколков Времени; О6 —
     ур.30 + условия финала.
4. **Навигация.** Тап по открытому острову → `Navigation.gotoIsland(id)` (рестарт
   IslandScene с этим островом). Каждый остров — своя сетка/сцена-ферма (task-05, 25).
5. **Сердце Архипелага.** Окутано вращающимся туманом, открывается последним при
   выполнении условий финала (Хроносаур + Астралон → Некзус).
6. **Множество островов = множество ферм.** Состояние построек/садов/драконов — на
   каждый остров своё (структура `SaveData` расширяется: `islands: { [id]: {...} }`).

## Definition of Done

- [ ] Карта показывает 6 узлов с корректным состоянием locked/unlocked.
- [ ] Закрытый остров под туманом; открытие снимает туман и анимируется.
- [ ] Открытие требует фрагментов + ресурсов + уровня; цена списывается.
- [ ] Переход между открытыми островами сохраняет состояние каждой фермы.
- [ ] Сердце открывается последним по условиям финала.
