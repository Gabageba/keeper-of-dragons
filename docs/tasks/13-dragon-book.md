# 13 — Книга Драконов (коллекция и прогресс)

**Фаза:** 4 — Цель и мотивация · **Зависит от:** 02, 03

## Цель

Главный коллекционный прогресс игры. Книга показывает все 57 видов: открытые — с
артом, лором, стихией, ресурсом и найденной комбинацией; неоткрытые — силуэтом.
Прогресс-бар коллекции. 100% = звание «Великий Хранитель» и секретный Вечный.

## Что делать

1. **DragonBookModal (React-модалка, `ui/modals/`).** Сетка карточек по редкостям (вкладки/секции: Обычные 7,
   Необычные 15, Редкие 12, Эпические 9, Легендарные 7, Мифические 4, Древние 3).
   Прогресс «открыто N / 57» сверху.
2. **Карточка вида.**
   - Открыт (`book_discovered.includes(id)`): арт, кличка вида, стихия, редкость,
     ресурс и произв./ч, любимый/нелюбимый биом и еда, **найденная комбинация**
     (родители из рецепта), биография (`bio`), бонус (для эпиков+).
   - Не открыт: тёмный силуэт + «?», но видна подсказка-рецепт (скрещивание
     прозрачно — GDD), чтобы игрок знал, к чему стремиться.
3. **Запись при открытии.** `discoverInBook(id)` вызывается из task-12 при первом
   вылуплении нового вида → анимация открытия страницы, награда (золото/титул).
4. **Фильтры/поиск.** По стихии, редкости, «есть/нет в наличии». Удобно при 57 видах.
5. **Связь с родословной (task-15).** Из карточки — переход к экземплярам этого
   вида и их дереву.
6. **100% коллекции.** При всех 56 видах → разблокировать Вечного (#57) и звание
   (task-16, task-22).

## Definition of Done

- [ ] Книга показывает 57 слотов, сгруппированных по редкости.
- [ ] Неоткрытые — силуэт с «?», открытые — полная карточка с лором и комбинацией.
- [ ] Прогресс-счётчик обновляется при открытии нового вида.
- [ ] Открытие нового вида проигрывает карточку и выдаёт награду.
- [ ] 100% коллекции триггерит разблокировку Вечного.
