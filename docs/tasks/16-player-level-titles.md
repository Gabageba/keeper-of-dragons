# 16 — Уровни Хранителя, опыт и звания

**Фаза:** 5 — Прогрессия и мир · **Зависит от:** 03

## Цель

Опыт даётся за сбор ресурсов, кормление, заказы и открытие видов. Новый уровень
разблокирует здания/острова/механики. Звания отражают прогресс.

## Что делать

1. **XP и уровни.** В `GameState`: `xp`, `player_level`. Кривая опыта в
   `data/levels.json` (XP до уровня N). `addXp(n)` копит, при превышении порога —
   `level++`, событие `level:up` → награда + всплывашка.
2. **Источники XP.** Сбор ресурсов (task-07), кормление (task-09), сдача заказов
   (task-14), открытие нового вида (task-12/13). Значения — баланс (task-26).
3. **Разблокировки по уровню (GDD «Прогрессия»).**
   - 1–2: старт, Эмберикс, первый сад, заказы;
   - 3–4: Зал Скрещивания, Библиотека, первые гибриды;
   - 5: Остров 2, Акваринт, Фростел, Лунный сад;
   - 6–8: Кузница, Верфь, экспедиции, Лаборатория;
   - 10: Остров 3, Вольтракс, тень; 12–15: гибриды 2 ур., Храм;
   - 16: Остров 4, Космос; 20–22: Остров 5, Хроносаур; 25–30: Остров 6, Некзус,
     Зал Хранителя, эндгейм.
   Таблица разблокировок — в `data/unlocks.json`, проверяется на `level:up`.
4. **Звания.** Ученик (1–5), Хранитель (6–15), Мастер Стихий (16–25), Архимаг
   Архипелага (26–30), Великий Хранитель (вся коллекция). Показывать в UI/профиле.
5. **UI.** Полоса опыта и уровень в UIScene (task-30). Всплывашка «Новый уровень!»
   со списком разблокированного.

## Definition of Done

- [ ] XP начисляется из всех источников, уровень растёт по кривой.
- [ ] Достижение уровня разблокирует соответствующий контент (по `unlocks.json`).
- [ ] Звание меняется по диапазонам уровней.
- [ ] Полоса опыта и уровень видны в UI и обновляются реактивно.
