# 11 — Зал Скрещивания: родители, растения, рецепты

**Фаза:** 3 — Скрещивание · **Зависит от:** 02, 10 · **Ядро игры**

## Цель

Игрок выбирает двух взрослых драконов и нужные растения, запускает скрещивание.
Через таймер получает яйцо по фиксированному рецепту `A + B = C` (никакой генетики).
Растения тратятся, родители заняты на время процесса.

## Модель данных

`BreedRecipe` (в `src/types`): `parent_1`, `parent_2`, `result`, `chance`,
`required_plants[]`, `boost_plant?`, `breed_time_hours`, `fail_result`.
`BreedingProgress { parent_1_uid, parent_2_uid, recipe_id, started_at, ready_at }`.

## Что делать

1. **BreedingScene (модалка).** Два слота-родителя (выбор из списка взрослых,
   свободных драконов) + слоты растений. По выбору пары:
   - `ContentLoader.findRecipe(a,b)` → показать требуемые растения и (если открыто
     в Книге) возможный результат. Скрещивание прозрачно — игрок видит результаты в
     Книге (GDD «Что упрощено»).
2. **Проверки запуска.** Оба дракона взрослые и свободны; есть требуемые растения в
   нужном количестве; есть свободный слот скрещивания (уровень здания, task-18).
3. **Запуск.** Списать растения; `breeding.active = { ..., ready_at = now +
   breed_time_hours×3600000 }`; пометить обоих родителей «заняты».
   - Ускорение на 20% при уровне 3 Зала / камнями (task-26).
4. **Результат по таймеру.** При `ready_at <= now`:
   - бросок шанса: с вероятностью `chance` (+ `boost_plant.bonus_chance`, если
     добавлено редкое растение) → яйцо `result`; иначе `fail_result`
     (`random_parent` = яйцо одного из родителей).
   - яйцо отправляется в Инкубатор (task-10); родители освобождаются.
5. **Нет рецепта.** Если пара не имеет рецепта — запретить или дать яйцо
   случайного родителя (по балансу). На MVP — запретить с подсказкой.
6. **Слоты.** Зал ур.2 — второй слот скрещивания; ур.3 — −20% времени.

## Definition of Done

- [ ] Эмберикс + Терравор + 5 Огнецвет + 5 Корень Силы → через 4 ч яйцо Лавакрона
      (с шансом 70%, +10% при Жарком Мхе).
- [ ] Растения списываются; родители заняты весь процесс; по завершении свободны.
- [ ] Неудача даёт `fail_result` (яйцо родителя), а не пустоту.
- [ ] Нельзя запустить без нужных растений/свободных драконов/слота.
- [ ] Результат корректен после оффлайна.
