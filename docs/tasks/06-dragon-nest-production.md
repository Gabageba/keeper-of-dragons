# 06 — Гнездо дракона и производственный цикл

**Фаза:** 1 — Ядро фермы · **Зависит от:** 05 · **Это «петля удовольствия» MVP**

## Цель

Взрослый дракон в гнезде производит ресурс **не пассивно**, а по циклу
«покормить → подождать → собрать → покормить снова». Чтобы дракон произвёл партию
ресурса, его нужно покормить; через `PRODUCTION_DURATION_MS` партия готова к сбору.
Над гнездом — индикатор состояния цикла. Это ядро всей экономики.

## Модель цикла

Состояния дракона (`PRODUCTION_STATE`):

- **HUNGRY** — нечего собирать, можно покормить.
- **PRODUCING** — партия зреет (`producing_until > now`).
- **READY** — партия готова (`pending_amount`, `producing_until <= now`).
- **FULL** — склад ресурса полон (`>= MAX_RESOURCE_PER_TYPE`), кормить нельзя.

Один батч за раз: пока партия не собрана, кормить нельзя.

## Что делать

1. **Объект гнезда.** `game/objects/NestSprite.ts`: спрайт гнезда (2×2), спрайт
   дракона, кличка, «пузырь» состояния цикла. Пузырь:
   - READY → зелёный, число партии, пульсирует;
   - PRODUCING → янтарный, минут до готовности;
   - HUNGRY → золотой «!» (можно покормить);
   - FULL → красный «МАХ», пульсирует.
2. **Размер партии (при кормлении).**
   ```
   N = yield_per_feed
       × (1 + yield_level × YIELD_UPGRADE_BONUS_PER_LEVEL)   // улучшение
       × foodMult        // любимая +food_bonus / нелюбимая −food_penalty / 1
       × biomeMult       // любимый биом +biome_buff / нелюбимый +biome_debuff / 1
   ```
   `N` фиксируется в момент кормления (`pending_amount`), не пересчитывается со
   временем. **Потолка-за-партию нет** — партия может быть любой.
   Расчёт — `feedYield()` в `store/slices/dragonsSlice.ts`.
3. **Таймер.** Кормление ставит `producing_until = now + PRODUCTION_DURATION_MS`
   (единая константа, еда на время не влияет). Готовность считается от timestamp —
   переживает оффлайн без накопления.
4. **Лимит склада.** Кормить нельзя, если `resource >= MAX_RESOURCE_PER_TYPE`
   (состояние FULL, «не голодный»). Сам **сбор** может «перелить» через лимит —
   готовая партия начисляется целиком (99 + 10 → 109). Исключение: бонус Вечнозима
   (`infinite_storage`).
5. **Только взрослые.** Детёныш (`stage !== 'adult'`) не производит (см. task-12).
6. **Глобальные/пассивные бонусы.** Драконы с `global_bonus` влияют на множители
   острова — заложить хук (детально в task-22/26).

## Definition of Done

- [x] Покормленный Эмберикс через `PRODUCTION_DURATION_MS` отдаёт партию ресурса.
- [x] Размер партии зависит от еды, улучшения и биома.
- [x] Готовность корректна после оффлайна (timestamp).
- [x] Кормление заблокировано, пока партия не собрана или склад полон.
- [x] Детёныш не производит ресурс.
