# 06 — Гнездо дракона и таймер производства

**Фаза:** 1 — Ядро фермы · **Зависит от:** 05 · **Это «петля удовольствия» MVP**

## Цель

Взрослый дракон в гнезде непрерывно производит свой ресурс со скоростью
`production_per_hour`. Над гнездом со временем появляется индикатор «есть что
собрать». Это ядро всей экономики.

## Что делать

1. **Объект гнезда.** `src/objects/Nest.ts` (Phaser.GameObjects.Container):
   спрайт гнезда (2×2 клетки, тематический по виду — task-27), спрайт дракона,
   кличка над гнездом, «пузырь» накопленного ресурса.
2. **Расчёт накопления (от timestamp, не от update).**
   ```
   elapsed_h = (now - last_collected) / 3_600_000
   rate = production_per_hour × (1 + biomeBuff) × (1 + foodBonusActive)
   accumulated = min(floor(rate × elapsed_h), storageCap)
   ```
   Бафы: биом (task-21), любимая еда (task-09). На MVP можно без бафов.
3. **Потолок склада.** Накопление останавливается на `storageCap` (баланс task-26).
   Над гнездом — индикатор «полно» (стимул заходить чаще). Исключение: бонус
   Вечнозима.
4. **Индикация.** Когда `accumulated > 0` — иконка ресурса всплывает над гнездом,
   при заполнении — пульсирует. Без поллинга каждый кадр: пересчёт по таймеру раз
   в несколько секунд + при заходе в сцену.
5. **Только взрослые.** Детёныш (`stage !== 'adult'`) не производит ресурс
   (см. task-12).
6. **Глобальные/пассивные бонусы.** Драконы с `global_bonus` (легендарные+) влияют
   на скорость/таймеры (Драконий Король +5% каждому и т.п.) — заложить хук, считать
   суммарный множитель острова (детально в task-22/26).

## Definition of Done

- [x] Гнездо с Эмбериксом за 1 час копит ~8 ресурса (проверка ускорением времени).
- [x] Накопление корректно после оффлайна (через task-04).
- [x] Накопление стопорится на потолке склада.
- [x] Детёныш не производит ресурс.
