# 00 — Каркас проекта (Phaser + TS + Vite)

**Фаза:** 0 — Фундамент · **Статус:** ✅ выполнено (скелет создан) · **Зависит от:** —

## Цель

Получить запускаемый каркас игры на Phaser 3 + TypeScript со сборкой через Vite,
единым кодбейсом для веба и (позже) мобилки через Capacitor.

## Что уже сделано (скелет в репозитории)

- `package.json` со скриптами `dev` / `build` / `typecheck` и зависимостями
  `phaser@^3.80`, `typescript`, `vite`.
- `tsconfig.json` — строгий режим (`strict`, `noUnused*`), alias `@/*→ src/*`.
- `vite.config.ts` — `base: './'` (важно для Capacitor/file://), alias, dev-сервер
  с `host: true` (тест с телефона в той же сети).
- `index.html` — контейнер `#game`, мета viewport с `viewport-fit=cover` и
  запретом зума (мобилки).
- `src/main.ts` + `src/config.ts` — точка входа и конфиг Phaser (Scale.FIT,
  логическое разрешение 1280×720, `TILE_SIZE = 64`).
- Заглушки сцен: `BootScene`, `PreloadScene`, `IslandScene`, `UIScene`.
- Стартовый JSON-контент в `public/data/`.

## Проверка готовности (Definition of Done)
