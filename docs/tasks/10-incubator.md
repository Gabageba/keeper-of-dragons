# 10 — Инкубатор и инкубация яиц

**Фаза:** 3 — Скрещивание · **Зависит от:** 05, 03

## Цель

Яйца (из скрещивания, заказов, экспедиций) помещаются в Инкубатор и вылупляются по
таймеру. Уровень здания определяет число одновременных слотов (1 → 3).

## Модель данных

`EggState { dragon_id, started_at, ready_at }`, массив `incubator: EggState[]`
(уже в `src/types`). Лимит слотов = функция уровня Инкубатора.

## Что делать

1. **Получение яйца.** Любой источник (скрещивание/заказ/экспедиция/магазин) даёт
   яйцо вида `dragon_id`. Если есть свободный слот — `started_at = now`,
   `ready_at = now + hatch_time_minutes×60000`.
   - Если слотов нет — яйцо в «ожидание» (или запретить, по балансу).
2. **Сцена/панель инкубатора.** Слоты с яйцом (цвета стихии), таймером и
   прогресс-кольцом. Готовое яйцо — пульсирует, кнопка «Вылупить» (→ task-12).
3. **Ускорение.** Инкубационный Порошок (−2 ч), Магические Камни (мгновенно — базовое
   яйцо за 5 камней), Цветок Времени (−30 мин). Бонус Прародителя: все яйца
   инкубируются на 30% быстрее (глобальный множитель).
4. **Таймер от timestamp.** Готовность = `ready_at <= now`. Завершение «задним
   числом» после оффлайна (task-04).
5. **Уровни инкубатора.** Ур.1 — 1 слот, макс — 3 одновременно (task-18).

## Definition of Done

- [ ] Яйцо Эмберикса вылупляется через 30 мин (или мгновенно за 5 камней).
- [ ] Нельзя инкубировать больше яиц, чем слотов по уровню.
- [ ] Таймер переживает перезагрузку и оффлайн.
- [ ] Ускорители (порошок/камни/цветок) уменьшают таймер.
