# 21 — Предпочтения биомов (бафы/дебафы)

**Фаза:** 5 — Прогрессия и мир · **Зависит от:** 06, 08, 05

## Цель

Лёгкий слой стратегии при расстановке: гнездо рядом с любимым биомом дракона даёт
+15% производства, рядом с нелюбимым — −10%, нейтральный — ±0. Складывается с
бонусом еды (до +35%).

## Правила (GDD «Предпочтения биомов»)

| Размещение | Эффект |
|---|---|
| 🟢 любимый биом | +15% производства, довольная иконка |
| ⚪ нейтральный | ±0% |
| 🔴 нелюбимый | −10%, грустная иконка |
| ✨ любимый биом + любимая еда | +35% суммарно |

**Перк высоких редкостей:** легендарные/мифические/древние — `adaptive: true` →
**нет штрафа** ни в одном биоме, в любимом всё равно +15%.

Базовые предпочтения по стихиям: Огонь 🟢Вулкан/🔴Лёд; Вода 🟢Лунный/🔴Вулкан;
Земля 🟢Кристал/🔴Гроза; Гроза 🟢Гроза/🔴Кристал; Лёд 🟢Лёд/🔴Вулкан; Ветер
🟢Воздух/🔴Кристал; Природа 🟢Природ/🔴Тень; Свет 🟢Природ-Свет/🔴Тень; Тень
🟢Тень/🔴Свет-Природ; Космос 🟢Космос/🔴Вулкан; Время+ — адаптивный. Гибриды
наследуют любимый биом одного из родителей. Конкретика — в `favorite_biome`/
`disliked_biome` дракона (Бестиарий).

## Что делать

1. **Определение «рядом с биомом».** Гнездо считается «в биоме», если соседствует
   (примыкает гранью/в радиусе) с биомом-садом на сетке. Определить правило и
   считать при размещении (task-05) → записать `DragonState.biome`.
2. **Множитель в производстве.** В расчёте (task-06):
   `biomeMult = adaptive ? (fav ? 1+biome_buff : 1) : (fav ? 1+biome_buff : disliked ? 1+biome_debuff : 1)`.
3. **Индикация.** Над гнездом — 🟢/⚪/🔴 иконка настроения. В панели дракона —
   строка «Биом … +15% 🟢».
4. **Подсказки при размещении.** При переносе гнезда показывать предполагаемый
   эффект рядом с каждым биомом (помощь в стратегии).

## Definition of Done

- [ ] Гнездо у любимого биома даёт +15%, у нелюбимого −10%, нейтрально ±0.
- [ ] Адаптивные (легендарные+) не получают штраф, но получают +15% в любимом.
- [ ] Биом-бафф складывается с бонусом еды (до +35%).
- [ ] Иконка настроения над гнездом и строка в панели отражают текущий биом.
