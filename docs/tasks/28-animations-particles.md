# 28 — Анимации и частицы

**Фаза:** 7 — Арт и полировка · **Зависит от:** 27

## Цель

Оживить ферму: idle-покачивание драконов, реакции на тап, эффект сбора, частицы
ауры редкости, переходы экранов. «Сочность» — то, ради чего возвращаются.

## Что делать

1. **Idle-анимации драконов.** Лёгкое покачивание/дыхание + моргание (tween или
   спрайт-лист 4–6 кадров — GDD: этого хватает для милого эффекта). Можно начать с
   tween по статичной póse.
2. **Реакция на тап.** Подпрыгивание (scale/y), «happy»-эмоут ❤ при кормлении/сборе
   (task-07/09).
3. **Эффект сбора ресурса.** Иконки летят по дуге к счётчику UI, счётчик «тикает»,
   мелкие частицы-блёстки.
4. **Аура редкости (частицы).** Phaser ParticleEmitter по `rarity`: редкие+ —
   искорки; мифические — переливающиеся частицы; древние — золотая аура. Единый
   менеджер (task-27).
5. **Биом-специфика.** Дымок над вулканическим садом, искры в грозовом, падающий
   снег в ледяном, светлячки в природном — атмосферные эмиттеры.
6. **Переходы сцен/модалок.** Fade/slide при открытии Книги/Рынка/Скрещивания,
   анимация открытия страницы Книги, тряска→трещина яйца при вылуплении (task-12),
   снятие тумана на карте (task-17).
7. **Производительность.** Пул частиц, лимиты эмиттеров, отключение анимаций вне
   экрана/при низком FPS (важно для мобилок).

## Definition of Done

- [ ] Драконы покачиваются/моргают в idle и реагируют на тап.
- [ ] Сбор ресурса визуально «летит» в счётчик.
- [ ] Аура редкости отличается между обычными/редкими/мифическими/древними.
- [ ] Вылупление, открытие Книги и снятие тумана анимированы.
- [ ] Стабильные ~60 FPS на среднем телефоне (или авто-деградация эффектов).
