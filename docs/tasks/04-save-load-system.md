# 04 — Сохранение/загрузка и оффлайн-прогресс

**Фаза:** 0 — Фундамент · **Зависит от:** 03

## Цель

Прогресс сохраняется локально (LocalStorage → позже IndexedDB), таймеры считаются
от системного времени (`Date.now()`), а при возвращении в игру корректно
начисляется накопленное за оффлайн время. Серверной валидации нет (пет-проект).

## Что уже есть

`src/systems/SaveManager.ts` — `load()`, `save()`, `createNew()`, `reset()`,
ключ `dragons_save_v1`, версия схемы.

## Что делать

1. **Связать с GameState.** При старте: `SaveManager.load() ?? createNew()` →
   передать в `GameState`. При мутациях стор помечает «dirty».
2. **Автосейв.**
   - по таймеру раз в ~10–15 c, если «dirty»;
   - на событиях `visibilitychange` (вкладка скрыта), `pagehide`, `blur` — критично
     для мобилок, где приложение сворачивают;
   - принудительно после важных действий (покупка, старт скрещивания).
3. **Оффлайн-начисление.** При загрузке для каждого взрослого дракона:
   `elapsed = now - last_collected` → накопленные ресурсы = `production/час × часы`
   (с учётом бафов биома/еды и потолка склада). Аналогично — завершились ли таймеры
   роста растений, инкубации, скрещивания, экспедиций (сравнить `ready_at <= now`).
   Показать игроку сводку «пока вас не было: +X ресурсов, вылупилось Y».
4. **Потолок склада.** Ресурсы копятся не бесконечно (если нет дракона Вечнозим с
   бонусом «бесконечный склад»). Лимит — параметр баланса (task-26).
5. **Миграции версий.** В `load()` — `switch (data.version)` с пошаговым апгрейдом
   старых сохранений. На MVP достаточно заглушки, но точку расширения завести.
6. **Защита от порчи.** `try/catch` вокруг `JSON.parse` (уже есть). Опционально —
   бэкап предыдущего снимка в отдельный ключ перед перезаписью.
7. **Переход на IndexedDB позже.** Когда снимок вырастет — вынести в IndexedDB
   (через `idb-keyval` или нативно). Интерфейс `SaveManager` оставить тем же.

## Definition of Done

- [ ] Перезагрузка страницы сохраняет прогресс (монеты, драконы, сады).
- [ ] Сворачивание/закрытие вкладки на мобилке не теряет прогресс.
- [ ] После 1 часа оффлайна дракон 8/ч даёт ~8 ресурсов (в пределах потолка).
- [ ] Таймеры скрещивания/инкубации/экспедиций завершаются «задним числом».
- [ ] Кнопка сброса (`SaveManager.reset`) доступна в dev-меню.
