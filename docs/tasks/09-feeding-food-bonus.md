# 09 — Кормление драконов и бонус любимой еды

**Фаза:** 2 — Сад · **Зависит от:** 06, 08

## Цель

Растения тратятся на кормление драконов. Кормление любимой едой даёт +20% к
производству (на время), нелюбимой — без бонуса/со штрафом. Кормление также
растит детёнышей (см. task-12).

## Что делать

1. **Действие «накормить».** В панели дракона — кнопка «Покормить» → выбор
   растения из инвентаря.
   - Любимая еда (`favorite_food`) → бафф `food_bonus` (+20%) к производству на
     заданное время (баланс task-26, напр. 1–4 ч).
   - Нелюбимая (`disliked_food`) → без баффа (или лёгкий штраф/дракон отказывается).
   - Прочая → насыщение без бонуса.
2. **Стек бафов.** Любимый биом (+15%) и любимая еда (+20%) складываются → до +35%
   (GDD). Передать множитель в расчёт производства (task-06).
3. **Таймер баффа.** Хранить `food_buff_until` в `DragonState`; в расчёте
   производства учитывать только период действия. После истечения — бафф спадает,
   иконка над гнездом гаснет.
4. **Кормление детёныша.** Детёнышу нужно `N` кормлений до взросления (3–5, task-12).
   Кормление любимой едой может ускорять/считаться полноценнее (по балансу).
5. **UX.** Подсказка «🟢 любит / 🔴 не любит» прямо в списке еды. Индикатор
   активного баффа над гнездом (как «happy»-эмоция в GDD).

## Definition of Done

- [ ] Кормление Эмберикса Лавовой Ягодой даёт +20% к ресурсу на время баффа.
- [ ] Биом-бафф и еда-бафф складываются (+35% при обоих).
- [ ] Бафф истекает по таймеру и переживает оффлайн (от timestamp).
- [ ] Растение списывается из инвентаря при кормлении.

  стек до +35%).
- «Растения по стихиям» — у кого что любимое.
